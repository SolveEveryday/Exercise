## 로직 설명/피드백 가이드라인을 만들자

### 로직 설명 가이드라인
* 자료구조를 사용했다면 사용한 이유
* 로직에 대해 그렇게 짠 이유
* 로직을 최대한 직관적으로 설명

### 로직 피드백 가이드라인
* 더 좋은 자료구조 추천
* 더 효율적인 로직
* 이유가 타당하지 않음을 지적
* 이렇게 하면 설명이 더 직관적일 것 같다는 제시
